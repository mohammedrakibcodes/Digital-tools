# DigiTools

A modern digital tools marketplace built with React, Tailwind CSS, and DaisyUI. Users can browse premium digital products, add products to a cart, manage selected items, and complete checkout with an interactive and responsive user experience.

## Live Website

🔗 https://digital-tools-rakib.netlify.app/

## Project Overview

DigiTools is a digital products marketplace where users can explore premium tools, add products to their cart, remove selected items, view total costs, and proceed to checkout. The application includes toast notifications, responsive layouts, and dynamic cart management.

---

## Technologies Used

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- DaisyUI
- React Toastify
- JSON

---

## Key Features

### Product Browsing
- View premium digital products in a responsive card layout.
- Each product displays price, features, tags, and icons.
- Products are loaded dynamically from JSON data.

### Cart Management
- Add products to cart.
- Remove individual products from cart.
- Dynamic cart count in navbar.
- Total price calculation.
- Proceed to Checkout clears the cart.

### Interactive User Experience
- React Toastify alerts for:
  - Add to Cart
  - Remove Product
  - Checkout Success
- Animated buttons and hover effects.
- Fully responsive design for mobile, tablet, and desktop.

## Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/mohammedrakibcodes/Digital-tools.git
```

### Navigate to Project Folder

```bash
cd Digital-tools
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
